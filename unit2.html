Copyright is a law that pertains to the property rights of the individual, it is the exclusive, assignable legal right given to the originator for a fixed number of years, in short the creator of a particular piece of work holds the right to say how others may use that work, however in the case of a website, there is no copyright as is the case with typical works intellectual property, such as books, songs, films etc. However it will protect some or all elements within a website, under the copyright, designs and patents act 1988, by which enumerated classes of work are stated to be eligible for protection. Some of these classes include:



Text - If completely original, and proven to be a product of the authors skill, labour and effort, as well as not being copied from any other source, text within a website is protected as a literary work or a collection of literary works, this extends not only to text, but to any other type of recorded symbolic code regardless of literary merit.



HTML - As it incorporates the website text which is protected if the conditions of text within a website are met, however the extension here will be seen in the websites meta-information, headings, paragraphs, image sources, positions, text characteristics, page divisions and others, however again for this condition to be met for protection, requisite originality must be proven.



CSS - an acronym for Cascading Style Sheets, which is responsible for the appearance of a webpage, applied to markup by web browsers in the process of rendering a web page, influencing elements such as positioning and spacing, colours of page elements, font type and size as well as style again for this condition to be met for protection, requisite originality must be proven.


Software code - similar to HTML, it is protected as a literary work More information can be found within the sources below.
